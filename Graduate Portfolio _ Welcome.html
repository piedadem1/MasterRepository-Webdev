Salesforce

Salesforce is a cloud-based customer relationship management solution that develops relationships between companies and customers. According to www.yescrm.org, Salesforce is the #1 customer relationship management platform in the world. What makes Salesforce so powerful? Salesforce provides a system that lets customers begin storing information immediately after purchase. No hookup. No software to install. Salesforce provides out-of-the-box solutions that let growing businesses easily implement cutting-edge technology and connect all of the solutions that are important to them. Salesforce was founded by founder Marc Benioff(CEO, The Man), Halsey Minor(Cofounder), and Parker Harris(Architecture).

HTML 5 & CSS 3

HTML 5: The use of HTML 5 has a better web standard than the previous versions. This has resulted in; HTMLs' ability to load faster, improve performance, use less bandwidth, and allow batteries to last longer on devices. CSS 3 is the latest evolution of the Cascading style sheets language which was published in June of 2011. CSS is used in describing the presentation of Websites, including colors, layout, and fonts. It allows the site to adapt the presentation to different types of devices, such as Desktops, Laptops or Ipads, or even printers. What makes CSS 3 great? CSS 3 uses the properties of CSS and developed them to include new features to provide ease of use to the designers. CSS 3 allows the designer to handle media queries and is capable of supporting responsive designs. CSS was proposed by Hakon Wium Lie who was working with Tim Berners-Lee(Founder of HTML).
